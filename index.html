JSoop

JSoop brings powerful OOP to your JS by providing the missing foundation for building sophisticated object oriented programs. It supplies all the pieces you need to start building right out of the box.

Classes

Everything you need to create classes, namespaces, advanced inheritance, access to parent methods, and apply mixins to your objects.

Mixins

Several power mixins are included that provide your objects with custom events, advanced configuration, and the ability to manage plugins.

Utility

Many utility functions including type checking, object cloning, namespacing, function binding, and much more.
