Utility

While utility functions are not the focus of JSoop, it does expose several of its internal utility methods.

Array

JSoop.util.Array.indexOf(haystack, needle, [startIndex])

This is a shim for Array.prototype.indexOf.

JSoop.toArray(array)

This converts whatever is passed in to an array. If the passed item is an array already, it will be returned. If it isn't, it will be wrapped in an array. This is useful for creating functions that can handle multiple items or a single item.



JSoop.each(array, fn, [scope])


This iterates over the array, executing the given function for each item. If the function returns false then the iteration will stop. If scope is defined, then the function will be bound to that scope. The function receives three arguments when called: the current item, the current index and the full array.



                
Function


JSoop.emptyFn()


You can use emptyFn as a placeholder function. Generally this is used to provide subclasses with template functions that can be overridden.


Jsoop.bind(fn, scope)


Bind allows you to create a version of a function that will always be bound to the given scope. Most of JSoop supports scoping, but when working with code not created by JSoop this can be very handy.
